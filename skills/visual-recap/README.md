# Apple Visual Recap

`/visual-recap` creates a local-first rendered review artifact for Apple
branches, commits, PRs, or working-tree diffs.

Use it after implementation when the developer needs to scan changed targets,
app surfaces, Swift diffs, verification evidence, and residual platform risks
before line-by-line review.

This skill does not replace code review. It structures the review surface and
keeps local annotations in sidecar files that an agent can read and consume.
