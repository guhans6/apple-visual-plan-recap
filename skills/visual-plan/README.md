# Apple Visual Plan

`/visual-plan` creates a local-first rendered plan for Apple development work.

Use it before implementation when the developer needs to review scope, affected
targets, Swift/SwiftUI/AppKit/UIKit surfaces, diagrams, verification evidence,
and open questions visually.

This skill does not generate production Apple UI. It creates planning MDX with
stable block IDs so comments can be anchored and read back by an agent through
local feedback sidecars.
