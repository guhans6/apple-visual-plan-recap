# Visual Companion Artifacts

This folder is the project-local home for visual companion artifacts.

The steady-state layout is:

```text
docs/visual-companion/
  README.md
  plans/
  recaps/
```

Use this space for the local companion review loop:

- forward plans
- implementation recaps
- evidence and related review assets
- local feedback sidecars that keep follow-up state readable and auditable

The local companion product is intentionally local-first and private by
default. The root prune effort keeps this folder as the durable artifact home
while older Apple-only and hosted-era repo framing is removed.
